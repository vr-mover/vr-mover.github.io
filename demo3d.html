📖 How to use ▼
  1. Set API key: open Settings → 🔑 quick-start picks a provider (some free) and autofills it; paste your key.
  2. Move around: WASD walks while the mouse is over the room; right-drag to look; scroll to step forward/back. The cursor stays free, so the panels are always clickable — no mode switching.
  3. Hover an object → its unified gizmo appears centred on it: drag a coloured arrow to move on that axis, a plane square to slide in that plane, a ring to rotate, an axis cube to scale that axis, or the white centre cube to scale uniformly. Click objects to select them — you can select several at once (each gets an outline + an o token); Esc clears the selection.
  4. Left-click any surface — floor, wall, ceiling or an object → here (a cursor showing the surface) · left-drag → a line with a direction arrow · Shift+left-drag → an area. Deleting a token removes its marker too.
  5. Click a catalog object to drop one in front of you, or ＋ Add object to build your own from a shape; Del removes the selected object.
  6. Type a command, then Send — the LLM streams back 3D (x/y/z) API calls.
  7. Gaze: the assistant tracks what you look at (see 👁 looking at…), so you can say “move this here” or “turn that toward me” and it resolves this/that from your view.

Welcome to the VR Mover 3D demo

Stand inside a room and tell an LLM how to rearrange the furniture around you. Walk about, select objects, point at the floor, and watch it stream back 3D API calls that move, rotate and scale things in real time — the same reasoning core as the VR system.

  • 🚶
    First-person navigation — WASD to walk, right-drag to look, scroll to step forward/back (touch: joystick + drag). The cursor stays free, so every panel is clickable.
  • 🎛️
    Gizmo editing — click an object to select it, then drag its arrows (move), rings (rotate) or cubes (scale). Objects always stay inside the room.
  • 🧱
    Add your own objects — build custom items from a cube, sphere, cylinder or cone; they join the LLM’s catalog instantly.
  • 🔑
    LLM API key — required to run the demo. The Settings → 🔑 quick-start lists providers (several with a free tier) and autofills everything. Your key stays in your browser only.
