📖 How to use ▼
  1. Set API key: open Settings → 🔑 quick-start picks a provider (some free) and autofills it; paste your key.
  2. Hover an object → gizmo: drag the centre square to move, the green forward handle to rotate (scroll wheel also rotates).
  3. Click empty floor → hit-point + h1 badge.
  4. Click object → select + o1; click again to deselect.
  5. Left-drag floor → line l1; right-drag → area a1.
  6. Hold right-click on object (~0.6 s) → delete (long-press on touch).
  7. Voice: Auto = continuous + auto-submit; Hold = push-to-talk.
  8. Delete a badge with ⌫ — the canvas marker is removed too.

Welcome to the VR Mover demo

A 2D, top-down reproduction of VR Mover: tell an LLM what to do with the furniture — by voice or text — while pointing, drawing lines and marking areas on the canvas, and watch it stream back API calls that rearrange the room in real time.

  • 🎙️
    Microphone — voice input needs the browser's mic permission; it is requested only when you press the mic button.
  • 🔑
    LLM API key — required to run the demo. No key yet? The Settings → 🔑 quick-start lists providers (several with a free tier), autofills the endpoint, and links you straight to the key page. Your key is kept in your browser's localStorage only and is sent directly to your chosen provider, nowhere else.
  • 🛋️
    Starter scene — the demo opens with a small living room you can rearrange right away. Use Clear scene for a blank floor, or Load example to bring it back.
  • 🧭
    Guided tour — ~1 minute, and now walks you through the gizmo (move & rotate), pointing gestures, and typing or speaking a command.
